## 6. drop-in replacement claim

### 6.1 the core claim

| claim | label | detail |
|-------|-------|--------|
| rhachet-brains-chutes is a "drop-in replacement" for rhachet-brains-togetherai | [SUMP] | true for the api contract and sdk pattern, but NOT true for: model availability (2 models absent), rates (differ per model), reliability (503 vs queue), and privacy (miners see prompts) |

### 6.2 what "drop-in" means vs what it doesn't

| aspect | drop-in? | detail |
|--------|----------|--------|
| openai sdk pattern | yes | same `baseURL` + `apiKey` override |
| `genBrainAtom` contract | yes | same `ask()`, `BrainSpec`, `episode` pattern |
| model catalog | **no** | 2 of 10 models absent; some ids differ |
| per-token rates | **no** | different rates per model; some cheaper, one costlier |
| reliability profile | **no** | 503 miner-dependent vs enterprise sla |
| prompt privacy | **no** | miners see cleartext vs known infrastructure |
