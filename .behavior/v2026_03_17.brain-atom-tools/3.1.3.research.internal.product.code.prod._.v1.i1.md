# 3.1.3.research.internal.product.code.prod

## summary

this research analyzes current production code patterns in rhachet-brains-chutes and classifies them for tool use implementation. patterns are marked as [REUSE], [EXTEND], or [REPLACE] based on how they map to tool use requirements.

### 3.3 cast function: tool messages [NEW]

**file**: `src/infra/cast/castIntoChutesToolMessages.ts`

**purpose**: convert `BrainPlugToolExecution[]` to openai message format for continuation

**xai reference**:
```ts
export const castIntoXaiToolMessages = (input: {
  executions: BrainPlugToolExecution[];
}): OpenAI.ChatCompletionMessageParam[] => {
  const messages: OpenAI.ChatCompletionMessageParam[] = [];
  messages.push({
    role: 'assistant' as const,
    content: null,
    tool_calls: input.executions.map((exec) => ({
      id: exec.exid,
      type: 'function' as const,
      function: {
        name: exec.slug,
        arguments: JSON.stringify(exec.input),
      },
    })),
  });
  for (const execution of input.executions) {
    messages.push({
      role: 'tool' as const,
      tool_call_id: execution.exid,
      content: JSON.stringify(execution.output),
    });
  }
  return messages;
};
```
