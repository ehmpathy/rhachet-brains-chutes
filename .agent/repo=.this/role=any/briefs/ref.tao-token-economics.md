## 3. token economics

### 3.1 emission schedule

| attribute | value |
|-----------|-------|
| daily emission | ~3,600 TAO (post dec 2025 halve) |
| pre-halve emission | ~7,200 TAO/day |
| halve schedule | bitcoin-like (periodic halves) |
| final supply | 21,000,000 TAO (hard cap) |

### 3.2 emission distribution

each block's tao emission is split:

| recipient | share |
|-----------|-------|
| miners (ai compute) | 41% |
| validators (quality score) | 41% |
| subnet owner | 18% |

### 3.3 what drives tao value

| factor | impact |
|--------|--------|
| **network usage** | more inference → more demand for subnets → more tao staked |
| **emission halves** | reduced supply → scarcity pressure |
| **subnet growth** | more subnets → more miners/validators → more tao locked |
| **speculation** | ai narrative, crypto market cycles |

## 4. recent developments (2025-2026)

| date | event |
|------|-------|
| dec 2025 | first emission halve (7,200 → 3,600 tao/day) |
| 2026 | subnet cap expansion (128 → 256 subnets) |
| 2026 | deutsche digital assets staked ETP on SIX swiss exchange |
| 2026 | dynamic tao (dtao) — subnet token price affects emission share |

### 4.1 dynamic tao (dtao)

the new economic model ties subnet emissions to subnet token prices:
- subnets with higher alpha token demand get more tao emissions
- replaces the old validator-controlled allocation
- more market-driven, less centralized
